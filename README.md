# Fusion Multi FPS
 基于Photon pun2 开发的多人fps游戏
游戏素材，模型，贴图来自商店或者自制，均有商用授权。

# 开发环境
unity 2020.3.26 + pun2 4.1.6.11

## demo游玩注意事项

++ 注册账号并登录
++ 单人模式尚未实装（2022.4.20）
++ 选择多人游玩模式
++ 选择游戏模式
++ 挑选一张地图
++ 输入房间名称
++ 输入人数限制
++ 创建房间进入部署界面
++ 挑选你喜欢的枪械
++ 叫上你的小伙伴一起来玩耍叭
