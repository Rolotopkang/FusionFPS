## demo游玩注意事项

+ 注册账号并登录
+ 单人模式尚未实装（2022.4.20）
+ 选择多人游玩模式
+ 选择游戏模式
+ 挑选一张地图
+ 输入房间名称
+ 输入人数限制
+ 创建房间进入部署界面
+ 挑选你喜欢的枪械
+ 叫上你的小伙伴一起来玩耍叭
